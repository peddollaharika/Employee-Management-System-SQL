# Employee Management System Analysis using SQL

## Project Overview
This project focuses on analyzing employee, payroll, department, qualification, bonus, and leave data using SQL. The objective is to extract meaningful insights that help in understanding workforce management, salary distribution, employee qualifications, leave patterns, and payroll processing.

## Tools Used
- MySQL
- SQL Queries
- ER Diagram
- Relational Database Design

## Database Tables
- Employee
- JobDepartment
- Qualification
- Leaves
- Payroll
- Salary_Bonus

## Analysis Performed
### Job & Department Analysis
- Number of job roles in each department
- Average salary by department
- Highest paying job role
- Department with highest salary allocation

### Qualification Analysis
- Employees with qualifications
- Qualification requirements by position
- Employees with highest qualifications

### Leave Analysis
- Employees taking leaves
- Department-wise leave patterns
- Leave impact on payroll

### Payroll & Compensation Analysis
- Total salary expenditure
- Monthly payroll processing
- Average bonus by department
- Highest bonus receiving department
- Average payroll after deductions

## Key Insights
- Finance department has the highest salary allocation.
- Finance Director is the highest-paid position.
- Legal department has the highest average salary and bonus.
- All employees possess at least one qualification.
- Leave distribution is balanced across departments.
- Total company salary expenditure is ₹43,21,000.
- Total monthly payroll processed is ₹27,78,000.
